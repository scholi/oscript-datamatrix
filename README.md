INSTALLATION

no installation needed. Just put the file datamatrix.py in your PYTHONPATH folder. To do so just do:
	cp datamatrix.py MYDEST/datamatrix.py
	echo 'export PYTHONPATH="${PYTHONPATH}:MYDEST"' >> ~/.bashrc
